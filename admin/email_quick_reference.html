Quick Setup (3 Steps)

Step 1: Get Gmail App Password
  • Google Account ‚Üí Security
  • Enable 2-Step Verification
  • Generate App Password
Step 2: Update Configuration
  • Edit admin/includes/email_config.php
  • Add your Gmail & App Password
Step 3: Enable & Test
  • System Settings ‚Üí Enable Email Alerts
  • Click "Test Email Configuration"

Email Types

Type Sent To When
Overdue User Equipment overdue
Borrow User Equipment borrowed
Return User Equipment returned
Low Stock Admin Stock below threshold

Configuration File

Location:

admin/includes/email_config.php

Required Settings:

SMTP_USERNAME = 'your-email@gmail.com'
SMTP_PASSWORD = 'your-app-password'
SYSTEM_EMAIL_FROM = 'your-email@gmail.com'
Important:
Use Gmail App Password, NOT your regular password!

Code Examples

1. Send Overdue Notification

require_once 'includes/email_config.php';

sendOverdueNotification(
  $conn,
  $user_id,
  'Arduino Kit',
  3 // days overdue
);

2. Send Borrow Confirmation

require_once 'includes/email_config.php';

sendBorrowNotification(
  $conn,
  $user_id,
  'Raspberry Pi',
  '2025-11-15' // expected return date
);

3. Send Return Receipt

require_once 'includes/email_config.php';

sendReturnNotification(
  $conn,
  $user_id,
  'Arduino Kit',
  'Good' // condition
);

4. Send Low Stock Alert

require_once 'includes/email_config.php';

sendLowStockAlert(
  $conn,
  'Arduino Kit',
  2, // current quantity
  5 // threshold
);

5. Check if Email Alerts Enabled

if (isEmailAlertsEnabled($conn)) {
  // Send email
  sendBorrowNotification(...);
}
